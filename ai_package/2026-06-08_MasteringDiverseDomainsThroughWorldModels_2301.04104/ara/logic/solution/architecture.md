真实环境交互(图像 64×64/向量观测 + 动作) -> 均匀回放缓冲区(在线队列; 存储潜在状态) -> 世界模型训练(RSSM: 编码器 CNN/MLP 将 x_t 映射为随机离散表示 z_t; Block-GRU 序列模型维护循环状态 h_t; 动力学预测器开环预测下一 z_t; 奖励/继续预测器从模型状态 {h_t, z_t} 预测 r_t 和 c_t; 解码器重建 x_t) -> 从回放轨迹起始状态出发在潜在空间展开 H=15 步想象轨迹 -> Actor(REINFORCE 策略网络 π_θ) 与 Critic(分布式价值网络 v_ψ, symexp twohot 参数化)并行在想象轨迹上训练 -> 三个组件(世界模型、Actor、Critic)并发梯度更新; 推理期仅由 Actor 从当前模型状态采样动作，无需向前规划
