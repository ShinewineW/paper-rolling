# Experiments

## E1: 图像标记器训练与DINO蒸馏语义效果验证
- **Verifies**: C1, C4
- **Setup**:
  - Model: 图像标记器（全卷积2D U-Net VQ自编码器，0.3B参数）
  - Hardware: 32 × A100 80GB GPU
  - Dataset: 4,700小时UK城市驾驶数据，图像分辨率288×512
  - System: AdamW优化器，学习率1e-4，权重衰减0.01，批大小160；损失项包含L1/L2/感知损失/GAN损失/codebook损失/DINO余弦相似度损失
- **Procedure**:
  1. 以288×512分辨率训练VQ自编码器，下采样因子D=16，词汇表大小K=8192，每帧编码为576个离散词元
  2. 加入DINO余弦相似度蒸馏损失，引导词元向预训练DINO特征对齐
  3. 训练200k步约4天，含5k步线性预热和10k步余弦衰减至最终学习率1e-5
  4. 通过PCA可视化对比基础VQ-GAN词元与DINO蒸馏词元的语义聚类效果（前3个主成分映射至RGB）
- **Metrics**: ['PCA词元嵌入可视化（定性）', '重建质量（定性）']
- **Expected outcome**: DINO蒸馏词元在PCA可视化中呈现出比基础VQ-GAN词元更清晰的语义聚类，同类物体（车辆、道路、天空）具有相似嵌入颜色
- **Baselines**: ['基础VQ-GAN词元（无DINO蒸馏）']
- **Dependencies**: ['预训练DINO自监督视觉Transformer']

## E2: 世界模型缩放定律验证实验
- **Verifies**: C2
- **Setup**:
  - Model: 参数量从0.65M到650M的系列小型自回归Transformer世界模型
  - Hardware: 未明确说明
  - Dataset: 与GAIA-1相同的驾驶数据集，含地理围栏验证集
  - System: 计算量估算：单词元前向-反向传播计算量C=6N（N为不含嵌入层的参数量），总计算量等于C乘以训练词元数
- **Procedure**:
  1. 训练参数量从0.65M到650M的系列模型（比GAIA-1世界模型小10,000x到10x）
  2. 在地理围栏验证集上测量各规模模型的验证交叉熵
  3. 拟合幂律函数f(x) = c + (x/a)^b至各数据点
  4. 外推预测6.5B GAIA-1模型的最终验证交叉熵并与实际结果比较
- **Metrics**: ['验证交叉熵', '幂律拟合外推精度（定性，Figure 8a）']
- **Expected outcome**: 幂律拟合可高精度预测GAIA-1最终验证交叉熵，且随模型规模增大损失持续下降，表明仍有较大改进空间
- **Baselines**: []
- **Dependencies**: ['GAIA-1完整6.5B模型作为外推目标验证点']

## E3: GAIA-1世界模型与视频解码器完整系统训练
- **Verifies**: C1, C3
- **Setup**:
  - Model: 世界模型（6.5B参数自回归Transformer）与视频解码器（2.6B参数3D U-Net扩散模型）
  - Hardware: 世界模型：64 × A100 80GB GPU；视频解码器：32 × A100 80GB GPU
  - Dataset: 4,700小时、约420M帧UK城市驾驶数据（2019-2023年，伦敦）
  - System: 世界模型：AdamW学习率1e-4、批大小128、DeepSpeed ZeRO-2+激活检查点、FlashAttention v2；视频解码器：AdamW学习率5e-5、批大小64、v参数化噪声预测目标
- **Procedure**:
  1. 世界模型在T=26帧、6.25Hz视频序列上训练，总序列长度15860；条件模式：无条件20%/动作条件40%/文本条件40%
  2. 世界模型训练100k步约15天，含2.5k步线性预热和97.5k步余弦衰减
  3. 视频解码器多任务联合训练：图像生成、视频生成、自回归解码、视频插值四任务等概率采样，训练300k步约15天
  4. 推理时世界模型输出以6.25Hz为基础，视频解码器两阶段时序上采样至25Hz
- **Metrics**: ['视频生成质量（定性）', '场景多样性（定性）', '条件控制准确性（定性）']
- **Expected outcome**: 系统可生成分钟级连贯驾驶视频，准确响应文本和动作条件，并在训练数据分布外的场景中保持合理行为
- **Baselines**: []
- **Dependencies**: ['E1（预训练图像标记器）', '预训练T5-large文本编码器（固定参数）']

## E4: 世界模型推理采样策略对比实验
- **Verifies**: C5
- **Setup**:
  - Model: GAIA-1世界模型（6.5B参数）
  - Hardware: 未明确说明
  - Dataset: 验证集驾驶视频
  - System: 自回归推理，每帧生成n=576个词元，词汇表大小K=8192
- **Procedure**:
  1. 对同一视频上下文分别应用argmax、全分布采样、top-k=50三种策略生成图像词元
  2. 绘制每种策略生成词元困惑度相对于词元位置（1至576）的变化曲线
  3. 与真实图像词元的困惑度分布进行对比（Figure 6）
- **Metrics**: ['词元困惑度分布（与真实图像词元困惑度分布的匹配程度）']
- **Expected outcome**: top-k=50采样的困惑度分布与真实词元最为接近；argmax困惑度极低（生成重复循环）；全分布采样出现极高困惑度尾部（出分布）
- **Baselines**: ['argmax采样', '全分布采样']
- **Dependencies**: ['E3（训练完成的世界模型）']

## E5: 多模态条件控制与涌现性质定性评估
- **Verifies**: C1, C3
- **Setup**:
  - Model: 完整GAIA-1系统（世界模型 + 视频解码器）
  - Hardware: 未明确说明
  - Dataset: UK城市驾驶数据验证集及分布外场景
  - System: classifier-free guidance（cosine衰减引导调度），top-k=50采样，DDIM采样器50步
- **Procedure**:
  1. 文本条件生成测试：使用天气/光照等文本提示生成驾驶场景（Figure 12）
  2. 动作条件生成测试：指定左转/强左转/右转/强右转等超出训练分布的自车动作（Figure 13）
  3. 多模态联合控制测试：同时使用文本和动作条件（文本指定前方公共汽车，动作指定超车）
  4. 多未来预测测试：从同一上下文反复采样生成多个不同合理未来（Figure 11）
  5. 长视频生成测试：从想象中生成数分钟的驾驶视频（Figure 10）
  6. 观察其他智能体对受控自车行为（如偏离车道）的反应性（Figure 13最后一行）
- **Metrics**: ['控制准确性（定性）', '场景多样性（定性）', '其他智能体反应合理性（定性）', '超出训练分布场景的泛化能力（定性）']
- **Expected outcome**: 模型准确执行文本和动作条件，生成训练数据中未曾出现的场景，并展现其他智能体的合理反应行为
- **Baselines**: []
- **Dependencies**: ['E3（完整训练系统）']
