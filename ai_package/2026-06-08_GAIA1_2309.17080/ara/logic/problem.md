# Problem Specification

## Observations

### O1: 自动驾驶需要对因车辆行为引发的各种可能未来结果进行精准预测，是安全规划与决策的核
- **Statement**: 自动驾驶需要对因车辆行为引发的各种可能未来结果进行精准预测，是安全规划与决策的核心需求
- **Evidence**: 摘要与引言指出「有效预测车辆行为可能引发的各种结果是关键难题」，且「精准未来预测使自动驾驶车辆能够预判并规划行动」
- **Implication**: 需要构建能捕获世界动态并支持可控预测的鲁棒世界模型

### O2: 现有世界模型依赖难以规模化获取的标注数据，且在模拟数据上训练的模型难以迁移至真实
- **Statement**: 现有世界模型依赖难以规模化获取的标注数据，且在模拟数据上训练的模型难以迁移至真实场景
- **Evidence**: 引言指出「这些方法通常依赖难以大规模获取的标注数据，在模拟数据上训练的模型可能无法完整捕获真实场景复杂性」
- **Implication**: 现有世界模型在真实自动驾驶场景中存在数据获取和表示泛化两方面的规模化瓶颈

### O3: 生成视频模型能生成逼真内容，但未能学到捕获预期未来事件的结构化世界动态表示
- **Statement**: 生成视频模型能生成逼真内容，但未能学到捕获预期未来事件的结构化世界动态表示
- **Evidence**: 引言指出「这类生成模型擅长生成视觉上令人信服的内容，但可能无法学习到对精确未来预测和鲁棒决策至关重要的演化世界动态表示」
- **Implication**: 纯生成视频方法缺乏可控、结构化的世界推理能力，无法直接支撑自动驾驶决策

## Gaps

### G1: 低维表示的世界模型难以生成高保真的未来预测样本
- **Statement**: 低维表示的世界模型难以生成高保真的未来预测样本
- **Caused by**: O2
- **Existing attempts**: []
- **Why they fail**: 低维隐变量表示能力有限，在复杂真实驾驶场景中生成预测帧缺乏足够细节，无法满足高保真要求

### G2: 生成视频模型缺乏结构化世界动态推理，无法实现动作/文本条件下的可控未来预测
- **Statement**: 生成视频模型缺乏结构化世界动态推理，无法实现动作/文本条件下的可控未来预测
- **Caused by**: O3
- **Existing attempts**: []
- **Why they fail**: 优化目标聚焦于视觉真实感而非世界状态转移的语义理解，无法在自车动作或文本条件下做出可解释的因果预测

## Key Insight
- **Insight**: 将视频帧离散化为语义VQ token，把世界建模统一为无监督下一token预测任务（继承LLM规模化优势），并以视频扩散解码器还原高保真像素，使单一框架同时具备世界模型的动态推理能力与生成模型的视觉真实感
- **Derived from**: ['G1', 'G2']
- **Enables**: 通过动作和文本条件生成多模态可控的驾驶场景视频，支持多样未来预测、场景理解涌现以及作为自动驾驶训练的神经模拟器

## Assumptions
- DINO蒸馏引导的VQ图像token能充分捕获驾驶场景语义，为世界模型的时序推理提供有效输入
- 速度和曲率两个标量（l=2）足以表达自车动作信息
- NLP领域的scaling laws可迁移至视频驾驶世界建模任务（论文第6节提供了实验支持）
