视觉观测经预训练 VAE 编码为潜变量 tokens，语言指令经 T5 编码器编码为语言嵌入 -> 输入 tokens 分为三组：干净第一帧潜变量 tokens（共享视觉锚点）、未来帧噪声视频 tokens（仅训练期存在）、动作 tokens -> Mixture-of-Transformer (MoT) 架构：视频分支（Wan2.2-5B 视频 DiT 骨干，约5B参数）与动作专家分支（hidden dim d_a=1024，约1B参数）通过结构化注意力掩码实现共享注意力，总参数约6B -> 所有 token 组通过 cross-attention 访问语言嵌入 -> 训练时，视频分支对未来帧噪声 tokens 执行流匹配去噪优化，动作分支对动作 tokens 执行流匹配去噪优化，两分支以干净第一帧 tokens 为共享视觉上下文，动作 tokens 不可访问未来视频 tokens -> 推理时，舍弃未来视频分支，仅干净第一帧潜变量 tokens 经视频 DiT 单次前向得到潜在世界表征 z(o,l)，动作专家 DiT 以该表征为条件执行10步流匹配去噪，直接输出动作序列 a_{1:H}，整体延迟约190ms
