多视角图像 -> vision encoder 提取 image tokens -> QT-Former 用 scene queries、perception queries、history queries 与 memory bank 聚合当前场景、交通元素和长期历史上下文 -> 两层 MLP 将 scene tokens 与 history tokens 映射到 LLM reasoning space -> LLM 结合用户指令执行 scene description、history information review、scene analysis、action reasoning，并生成 planning token s -> generative planner 将 planning token 条件化到 latent space，与 ground-truth trajectory 的 latent distribution 对齐 -> GRU decoder 从 latent space 解码 multi-modal trajectory。训练期采用 QT-Former 辅助任务、LLM 自回归交叉熵和 generative planner 规划损失联合优化；推理期由 planning token 条件化生成轨迹，不把只用于推理选择的任何额外加权项写入训练目标。
