历史多视角相机观测与历史 ego 轨迹 -> History Encoder W_E 提取多视角几何特征并转换为 BEV embeddings -> Memory Queue w_M 对历史 BEV embeddings 做 Semantic- and Motion-Conditional Normalization, 用语义预测、ego-pose transformation 和 3D backward centripetal flow 生成仿射调制参数并累积历史记忆 -> World Decoder W_D 以 learnable BEV queries 为输入, 依次经过 deformable self-attention、temporal cross-attention、conditional cross-attention 和 feedforward network, 在动作条件约束下自回归生成未来 BEV embeddings -> prediction heads 通过 channel-to-height 输出未来 semantic occupancy 和 3D backward centripetal flow -> action-controllable generation 场景中丢弃 planner P, 直接用 velocity、steering angle、trajectory 或 commands 作为条件生成未来状态 -> end-to-end planning 场景中 occupancy-based planner P 基于 agent-safety cost、road-safety cost 和 learned-volume cost 选择候选轨迹, 再用 BEV Refinement 细化轨迹, 并把预测轨迹作为下一步 action condition 进行连续 rollout。
