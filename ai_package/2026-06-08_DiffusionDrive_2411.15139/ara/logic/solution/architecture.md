传感器输入(相机和/或激光雷达) -> 骨干网络(ResNet-34 或 ResNet-50) -> 感知模块(3D目标检测、BEV语义分割,nuScenes上额外含地图构建与运动预测) -> 提取条件信息z(BEV特征或PV特征 + 智能体查询 + 地图查询) -> 预计算K-Means锚点集合 $\{\mathbf{a}_k\}_{k=1}^{N_{\mathrm{anchor}}}$ -> 截断扩散前向过程(在每个锚点附近添加小量高斯噪声,生成锚定高斯分布样本 $\{\tau_k^i\}$) -> 级联扩散解码器(每层依次执行: 可变形空间交叉注意力与BEV/PV特征交互 + 智能体/地图查询交叉注意力 + FFN + 时间步调制层 + MLP预测置信分 $\hat{s}_k$ 与坐标偏移 $\hat{\tau}_k$,参数在各去噪步间共享,级联堆叠若干层) -> 推理时以DDIM更新规则迭代去噪(默认2步) -> 按最高置信分选取最终输出轨迹
