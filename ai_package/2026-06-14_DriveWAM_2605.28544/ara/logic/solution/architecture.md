相机图像、ego actions 与 ego states 被切成连续 chunks -> 每个已观测视频 chunk 由 pretrained VAE 编码为视频 latent，每个 action chunk 由 MLP action encoder 编码为 action token -> 视频 token 与 action token 按时间顺序组成统一 causal history，并注入 ego state 与 chunk-specific VLM guidance -> shared pretrained flow-matching video diffusion transformer 先对下一 chunk 的 future video latent 做 rectified-flow velocity prediction -> action 分支在同一 transformer 上以 future world latent、history、ego state、guidance 为条件做 inverse-dynamics action velocity prediction -> 训练期用 causal teacher-forcing mask 在 full clip 中并行去噪，推理期逐 chunk rollout，先采样 future video latent，再采样 action chunk -> 长时推理中用 selective KV memory 维护 video 与 action 的 bounded modality-aware memory pools。
