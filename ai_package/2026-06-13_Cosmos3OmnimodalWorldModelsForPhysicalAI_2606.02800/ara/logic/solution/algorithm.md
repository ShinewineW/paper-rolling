论文未给出显式损失公式。训练期目标需要严格分开：Reasoner 使用 next-token prediction objective，论文另述训练基础设施支持 cross-entropy loss for Reasoner text；Generator 使用 rectified flow matching objective across all modalities，构造 noisy latent，并让 denoiser 预测 constant velocity，使用 masked mean-squared error，clean conditioning tokens 不计入损失；mid-training 仍使用 rectified flow objective，总损失按模态 velocity MSE 加权求和，action loss 使用额外缩放以补偿 normalized action vectors 的较小 per-element MSE。推理期不是训练目标：语言 token 通过 next-token prediction 生成，图像、视频、音频、动作通过 diffusion loop 迭代去噪；classifier-free guidance、CFG parallelism、two-weight classifier-free guidance、denoising steps、guidance scale、time shift、skip video-latent decoding 都是采样或服务期策略，不应写入训练损失。
