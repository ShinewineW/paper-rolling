# Claims

## C1: V-M-C架构将世界模型与控制器解耦，实现压缩时空表示驱动的强化学习
- **Statement**: 将智能体分解为视觉模块V（VAE）、记忆模块M（MDN-RNN）和控制器C（线性层），以无监督方式快速训练大容量世界模型，再用参数极少的控制器利用其表示完成强化学习任务，从而绕开信用分配问题对大型网络训练的瓶颈。
- **Status**: 实验验证
- **Falsification criteria**: 若世界模型提取的特征与手工设计特征或其他压缩表示相比无性能优势，则该架构分离设计的必要性不成立。
- **Proof**: [E1, E2]
- **Evidence basis**: ['E1', 'E2']
- **Interpretation**: 大参数量（VAE+MDN-RNN合计数百万参数）集中在以无监督目标预训练的世界模型中，控制器C仅有数百至千余参数，使CMA-ES等黑盒优化方法可行，同时继承了大模型的表示能力，避免了深度RL在高维模型上的信用分配困难。
- **Tags**: ['improvement', 'descriptive']

## C2: 完整世界模型特征（zₜ与hₜ拼接）首次使CarRacing-v0任务得到解决
- **Statement**: 仅使用VAE空间特征zₜ的控制器（含或不含隐藏层）均无法达到CarRacing-v0的解任务阈值（100次平均分900），而同时使用zₜ和MDN-RNN隐状态hₜ的完整世界模型控制器达到新最优性能，并据论文所述首次解决该任务。
- **Status**: 实验验证
- **Falsification criteria**: 若存在不使用MDN-RNN隐状态的方法同样达到100次平均分900以上，则MDN-RNN隐状态的必要性主张不成立。
- **Proof**: [E1]
- **Evidence basis**: ['E1']
- **Interpretation**: hₜ压缩了历史动作与观测信息，并对未来zₜ₊₁的概率分布进行编码，使控制器能够「本能地」预判环境动态；而仅依赖zₜ的控制器缺乏时序预见性，在急弯处产生抖动和失控。
- **Tags**: ['improvement', 'causal']

## C3: 在MDN-RNN构建的梦境虚拟环境中训练的策略可成功迁移至真实VizDoom环境
- **Statement**: 以MDN-RNN为核心构建的虚拟OpenAI Gym环境（DoomRNN）可完全替代真实VizDoom环境进行策略训练；梦境中习得的策略部署到真实环境后，存活时步数远超解任务阈值（750步），且超越了已知排行榜最优成绩。
- **Status**: 实验验证
- **Falsification criteria**: 若梦境训练策略在真实环境中无法超过随机策略或解任务阈值，则该策略迁移性声明不成立。
- **Proof**: [E2]
- **Evidence basis**: ['E2']
- **Interpretation**: 梦境环境仅在潜空间中运行，无需渲染像素帧，计算效率更高；由于世界模型包含近似误差和随机噪声，梦境环境更具挑战性，使在其中训练的智能体在真实的、更稳定环境中反而表现更优。
- **Tags**: ['generalization', 'causal']

## C4: 提高温度τ可抑制对抗性策略但过高会使梦境训练失败
- **Statement**: MDN-RNN是真实环境的近似概率模型，智能体可在梦境中发现对抗性策略（如迫使怪物永不发射火球），这些策略在真实环境中无效。通过提高τ增大梦境随机性可抑制此类对抗性利用，但τ过高会使梦境环境过难，导致控制器无法习得有效策略，真实环境得分随之下降。
- **Status**: 实验验证
- **Falsification criteria**: 若极低τ设置下梦境训练的策略仍能良好迁移，或极高τ下策略仍能达到解任务要求，则该因果关系不成立。
- **Proof**: [E3]
- **Evidence basis**: ['E3']
- **Interpretation**: τ是梦境现实性与可利用性权衡的关键超参数。τ极低时（如0.1）模型趋近确定性LSTM，发生模式坍塌，怪物不再发射火球，梦境得分虚高但真实得分接近随机策略；中间τ（如1.15）时真实得分最高；τ过高时梦境过难，智能体甚至无法学习基础避险行为。
- **Tags**: ['causal', 'scoping']

## C5: 时序记忆（MDN-RNN隐状态hₜ）对高性能驾驶策略不可缺少
- **Statement**: 在CarRacing-v0中，将控制器限制为仅访问VAE空间特征zₜ而不使用MDN-RNN的hₜ，即使在控制器中添加隐藏层也无法达到解任务阈值；加入hₜ后性能显著提升并首次解决该任务。
- **Status**: 实验验证
- **Falsification criteria**: 若某不依赖RNN隐状态的方法也能在CarRacing-v0上达到100次平均分900，则时序记忆的必要性声明不成立。
- **Proof**: [E1]
- **Evidence basis**: ['E1']
- **Interpretation**: hₜ汇集了历史动作与观测的压缩记忆以及对下一帧概率分布的预测，为控制器提供了前瞻性信息；仅凭单帧空间特征zₜ，控制器只能做出基于当前帧的被动反应，缺乏对急弯等未来状态的预判，导致驾驶不稳定。
- **Tags**: ['causal', 'descriptive']
