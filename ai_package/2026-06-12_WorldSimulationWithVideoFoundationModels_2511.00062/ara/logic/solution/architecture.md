数据侧先经 shot-aware video splitting -> GPU-based transcoding -> video cropping -> 多阶段 filtering -> captioning -> semantic deduplication -> sharding，形成高质量、带 caption、去重且按语义组织的视频训练数据；模型侧将图像或视频经 WAN2.1 VAE 编码到 latent，再做 patchification；文本由 Cosmos-Reason1 编码，跨多个 blocks 拼接 token 激活并投影为文本 embedding；DiT velocity prediction network 在 latent 空间中交替使用 self-attention、cross-attention 和 feed-forward MLP，并由 timestep 的 adaptive layer normalization 调制；同一 Cosmos-Predict2.5 支持 Text2World、Image2World、Video2World，Image2World 和 Video2World 通过 frame-replacement 保持条件帧；训练流程为 progressive pre-training -> domain-specific SFT -> cooldown -> model merging -> RL post-training -> timestep distillation；应用扩展包括 Cosmos-Transfer2.5 control-net style 分支、多视角 world scenario map 控制、camera trajectory 控制和 action embedder MLP 条件化。
